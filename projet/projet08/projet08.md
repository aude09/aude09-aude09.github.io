# Simulation of a miniature autonomous car in Isaac Sim

## Context
Project carried out with **Isaac Sim** to explore the simulation of mobile robots in a 3D environment.  
The vehicle is equipped with a simulated camera that can detect and avoid obstacles.

## Objectives
- Understand the simulation of a mobile robot in a 3D environment.
- Implement a virtual camera to detect obstacles.
- Test different navigation and control scenarios.
- Optimize the 3D model for better performance.

## Skills highlighted
- Robotic simulation with Isaac Sim
- Vehicle control programming
- Visual perception and obstacle detection
- Compression and optimization of 3D models

## Project description
- The car moves in a simulated environment with obstacles (cubes, cylinders).  
- The simulated camera detects obstacles and adapts the vehicle's trajectory.  
- Compressed 3D model to optimize simulation performance.  
- Remote-controlled interaction to test different scenarios.
